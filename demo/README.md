# Demo — Drone vs Bird Live Detection

## How to run the live demo

1. Open `FINAL_drone_vs_bird.ipynb` in Google Colab.
2. Mount Google Drive and confirm the model weights path (see step below).
3. Run the **Demo** section at the bottom of the notebook.
